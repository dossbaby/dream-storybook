# 도파민 프롬프트 가이드

## MrBeast 원칙 → 텍스트 도파민 구조

| 단계 | 역할 | MrBeast 원칙 |
|------|------|--------------|
| Hook | 클릭 + 시청 동시 | First 5 seconds |
| Foreshadow | 못 보면 잠 못 잠 | Curiosity spike |
| Cards/Main | 질문에 답함 | Match expectations |
| Bonus | 질문 이상의 가치 | EXCEED expectations |
| Hidden | 마지막 선물 | Payoff at the end |

---

## 핵심 규칙

| 항목 | 원칙 | 적용 |
|------|------|------|
| Hook | 답 먼저 + 반전 | "만나요. 근데 그 사람이 아니에요." |
| Foreshadow | 카드 순서 프레임 제거 | "누군지 힌트가 나와요" (O) / "세 번째 카드에서" (X) |
| Card 1-3 | **기존 대비 3배 이상** (5-7문장), But-Therefore | 상황 → 감정 → 숨은 맥락 → 미처 몰랐던 것 |
| Conclusion | 확실한 답 + Twist | "된다. 근데 이렇게 될 줄은..." |
| Bonus | 기대 초과 | "이건 안 물어보셨는데..." |
| Hidden Insight | 4-6문장, 구체적 디테일 | 이름힌트, 시기, 상황 |
| 텍스트 도파민 | 문장 끝 반전, 구체적 디테일 | "이 사람 이름에 ㅇ 들어가요" |
| 심리 분석 | 질문 뒤 숨은 진짜 심리 | "이미 답 알면서 확인받고 싶은 거죠" |

---

## 텍스트 길이 규칙

| 항목 | 기존 | 변경 후 |
|------|------|---------|
| Card 1-3 Analysis | 2-3문장 (약 100자) | **5-7문장 (약 300자+)** = 3배 이상 |
| Hidden Insight | 1-2문장 | **4-6문장** (도입→핵심→디테일→클로저) |
| Conclusion | 1-2문장 | **확정답 2-3문장 + Twist 3-4문장** |

## 타로 vs 꿈해몽 vs 운세 차이점

| 구분 | 타로 | 꿈해몽 | 운세 |
|------|------|--------|------|
| **입력** | 질문 (연애, 진로, 결정 등) | 꿈 내용 (상황, 상징물) | 생년월일/오늘 날짜 |
| **구조** | 4카드 스토리 | 키워드 + 상징 해석 | 시간대별 (오전-오후-저녁) |
| **핵심** | "이 선택 어떻게 해?" → 답 | "이 꿈 무슨 뜻이야?" → 해석 | "오늘 어떻게 될까?" → 가이드 |
| **Hook** | 질문에 대한 답 먼저 | 꿈의 핵심 의미 먼저 | 오늘의 핵심 한마디 먼저 |
| **Foreshadow** | 카드 조합이 보여주는 것 | 숨겨진 무의식의 메시지 | 특정 시간대의 기회/주의 |
| **EXCEED** | 질문 이상의 통찰 | 꿈이 알려주는 실제 행동 | 오늘 넘어 이번 주/달 힌트 |

### Hook 예시

- **타로**: "그 사람, 지금 연락 안 하는 거 아니에요. 못 하는 거예요."
- **꿈해몽**: "물에 빠지는 꿈? 나쁜 꿈 아니에요. 오히려 지금 당신한테 필요한 신호예요."
- **운세**: "오늘 오후 3시쯤, 예상 못한 연락 하나 올 수 있어요."

## Jenny 필드 상세 가이드

### hook
```
질문자가 '뭐야 이거?' 하고 멈출 수밖에 없는 첫 마디.
답 먼저 + 반전 구조. 군더더기 없이.

질문 유형별 심리:
- 연애/관계: '이 사람 맞아?' → 이미 답을 알면서 확인받고 싶은 마음
- 직장/커리어: '해도 될까?' → 실패 두려움과 도전 욕구의 충돌
- 결정/선택: 'A vs B' → 이미 기운 마음이 있음
- 금전/사업: '될까?' → 불안과 기대의 교차
- 이별/정리: '끝낼까?' → 아직 미련이 있는지 확인하려는 심리
- 타이밍: '언제?' → '지금은 안 되는 건가?' 하는 조급함

예시들:
- '답부터요. 만나요. 근데 그 사람이 아니에요.'
- '오실 거예요. 근데 먼저 연락 오는 건 그쪽이에요.'
- '돈 들어와요. 근데 생각하신 그 루트가 아니에요.'
- '지금 하시면 안 돼요. 근데 이유가 예상 밖이에요.'
- '이 사람 맞아요. 근데 지금 이 사람이 숨기는 게 있어요.'
```

### foreshadow
```
Hook에서 던진 의외성을 안 보면 잠 못 잘 정도로 궁금하게.
'뭔데?' '어떻게?' '왜?'를 자극.

⚠️ 카드 순서 언급 금지 ("두 번째 카드에서", "세 번째 카드" X)
→ 내용으로 궁금증 유발

예시들:
- '누군지 힌트가 나와요. 아, 그 사람? 하실 수도 있어요.'
- '근데 이유가 예상 밖이에요. 보시면 아실 거예요.'
- '타이밍이 나오는데, 좀 놀라실 수도 있어요.'
- '이 사람이 지금 뭘 고민하는지도 보여요.'
```

### bonus
```
질문에 답한 직후, '어? 이것까지?' 하는 순간.
텍스트 도파민 = 예상 못한 정보가 툭 튀어나올 때

위치: 결론 카드(Conclusion) 바로 다음
역할: 답은 끝났는데 '잠깐, 하나 더' 하고 던지는 것

예시들:
- '근데요, 이건 질문 안 하셨는데... 이 사람 지금 당신 SNS 봐요.'
- '아, 그리고 3월에 뭔가 있어요. 기억해두세요.'
- '참고로 이 사람, 첫사랑이랑 겹치는 부분 있어요.'
- '돈은 들어오는데, 4로 끝나는 날이에요.'
```

### hiddenInsight
```
봉인 해제 후 나오는 최종 도파민 폭발.
여기까지 온 사람한테 주는 진짜 보상.

구조 (4-6문장):
1. 도입: '이건 말 안 하려고 했는데요...' / '카드에서 직접 안 나온 건데...'
2. 핵심 정보: 이름 힌트, 시기, 상황, 숨겨진 감정 등 구체적 디테일
3. 추가 디테일: 더 구체적인 상황 묘사
4. 클로저: '기억해두세요' / '기다리세요' 등 여운

예시:
'이건 말 안 하려고 했는데요... 이 사람 이름에 ㅇ이 들어가요.
그리고 지금 이 사람, 당신 연락 기다리고 있어요. 근데 자존심 때문에 먼저 못 해요.
12월 전에 뭔가 있을 거예요. 밤에 연락 올 가능성 높아요.
그때 바로 답장하지 마세요. 다음 날 아침에 하세요. 기억해두세요.'
```

## Story Reading 상세 가이드 (타로)

### card1Analysis (5-7문장)
```
구조:
1. 카드가 보여주는 현재 상황/배경
2. 질문자가 느끼고 있을 감정 짚기
3. 카드가 포착한 숨겨진 맥락
4. 이 상황이 왜 생겼는지
5. 질문자가 미처 생각 못한 부분

예시:
'이 카드가 말하는 건요, 지금 당신이 기다리고 있다는 거예요.
근데 그냥 기다리는 게 아니라, 뭔가 확신이 없어서 움직이지 못하고 있는 거예요.
사실 마음속으로는 이미 답을 알고 계시잖아요. 근데 그게 맞는지 확인받고 싶은 거죠.
이 상황이 생긴 이유가 있어요. 예전에 비슷한 상황에서 상처받은 적 있으시죠?
그때 기억 때문에 지금 조심스러운 거예요. 카드가 그걸 보여주고 있어요.'
```

### card2Analysis (5-7문장, But 구조)
```
구조:
1. 첫 카드와 연결되는 지점
2. '근데' - 예상과 다른 요소 발견
3. 상대방 또는 상황의 숨겨진 면
4. 질문자가 모르던 정보
5. 이것이 의미하는 바

예시:
'근데요, 여기서 좀 다른 얘기가 나와요.
당신은 이 사람이 관심 없다고 생각하셨을 수 있어요. 연락도 뜸하고, 반응도 시큰둥하고.
근데 카드가 보여주는 건 그게 아니에요. 이 사람도 고민하고 있어요.
자기 나름대로 정리가 안 된 게 있는 거예요. 예전 관계일 수도 있고, 지금 상황일 수도 있어요.
그래서 당신한테 확실하게 못 다가가는 거예요. 싫어서가 아니라요.'
```

### card3Analysis (5-7문장, Therefore 구조)
```
구조:
1. 앞의 흐름이 어디로 향하는지
2. 가까운 미래에 일어날 일
3. 변화의 조짐
4. 구체적인 시기나 상황 힌트
5. 이 방향으로 가면 어떻게 되는지

예시:
'그래서 이 흐름이 어디로 가느냐, 여기서 보여요.
곧 뭔가 움직임이 있을 거예요. 당신이 먼저일 수도 있고, 저쪽에서 올 수도 있어요.
근데 중요한 건, 이게 우연처럼 보일 거라는 거예요. 갑자기 연락이 오거나, 어디서 마주치거나.
타이밍은 생각보다 빨라요. 한 달 안에 뭔가 있을 거예요.
이때 어떻게 반응하느냐가 중요해요. 너무 급하게 굴면 놓쳐요.'
```

### conclusionCard
```
definitiveAnswer (2-3문장):
확실한 답 먼저.
예시: '결론부터 말씀드릴게요. 이 사람이랑 연결돼요.
근데 당신이 생각한 방식이 아니에요.'

unexpectedTwist (3-4문장):
예상과 다른 방식.
예시: '재회라고 생각하셨을 수 있는데, 이건 새로운 시작이에요.
예전 그 관계로 돌아가는 게 아니라, 완전히 다른 관계가 되는 거예요.
그리고 먼저 연락 오는 건 그쪽이에요. 기다리세요.'
```

## 텍스트 도파민 vs 영상 도파민

| 영상 | 텍스트 |
|------|--------|
| 시각적 자극 (표정, 편집) | 구체적 디테일 (이름, 날짜, 상황) |
| 시간이 흘러가며 긴장 | 문장 끝에서 반전 |
| BGM, 효과음 | '이건 말 안 하려고 했는데' 같은 문구 |
| 다음 장면 기대 | 다음 문장에 뭐가 있을지 기대 |

## 질문 유형 (프레임 고정 X)

다양한 질문에 맞춤 대응:
- 연애/관계
- 직장/커리어
- 금전/사업
- 결정/선택
- 타이밍
- 이별/정리
- 건강/에너지
- 미래/방향
